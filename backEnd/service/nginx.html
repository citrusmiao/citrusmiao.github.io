    # windows 下使用

    • 下载地址 (opens new window) 选择 Stable version 版本下载
    • 解压 nginx-1.22.1.zip
    • cmd 进入解压后的目录 例：
    PS E:\it\nginx-1.22.1>
    • 启动 nginx
    PS E:\it\nginx-1.22.1> start nginx
    • 快速停止
    PS E:\it\nginx-1.22.1> .\nginx -s stop
    • 完整有序的关闭
    PS E:\it\nginx-1.22.1> .\nginx -s quit

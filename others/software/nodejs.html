# 安装步骤

1.下载

2.安装

  • 安装地址 例：E:\node

3.配置

  • 在安装目录下新建2个文件夹 node_global 和 node_cache

  • 启动cmd，依次输入：

npm config set prefix "E:\node\node_global"
​npm config set cache "E:\node\node_cache"

4.系统环境变量 path 添加 E:\node\node_global

5.检查是否配置成功

npm root -g
  • 配置淘宝源
npm config set registry https://registry.npm.taobao.org

# 安装 node-sasss 报错 需要安装 python2 解决办法

npm i --ignore-scripts
  • node 14版本 对应 node-sass 4.14.1 版本 sass-loader 8.0.2 版本
npm i node-sass@4.14.1 --save-dev   
npm i sass-loader@8.0.2 --save-dev

# 安装配置 pnpm

  • 安装
npm install -g pnpm
  • 配置 pnpm 全局安装路径, 例：E:\it\nodejs16\.pnpm-store
pnpm config set global-bin-dir "E:\it\nodejs16\.pnpm-store"
pnpm config set cache-dir "E:\it\nodejs16\.pnpm-store\cache"
pnpm config set state-dir "E:\it\nodejs16\.pnpm-store\state"
pnpm config set global-dir "E:\it\nodejs16\.pnpm-store\global"
pnpm config set g-dir "E:\it\nodejs16\.pnpm-store"
  • 验证设置是否正确:
pnpm c get

  • 在系统环境变量 path 中添加 E:\it\nodejs16\.pnpm-store

  • 检查

pnpm c list
  • 添加 aliyu nnpm repo
pnpm set registry "https://registry.npmmirror.com"

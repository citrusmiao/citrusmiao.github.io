# 安装步骤

  • 安装 Python3.x 环境

  • 把pip 升级到最新版本

python.exe -m pip install --upgrade pip  
pip3 install --upgrade pip
  • 安装 jupyter
pip3 install jupyter
  • 运行 默认端口 8888
jupyter notebook
  • 运行 指定端口 9999
jupyter notebook --port 9999
  • 修改默认文件路径
jupyter notebook --generate-config
  • 在 C:\users\citrus\.jupyter 文件夹下找到 jupyter_notebook_config.py 配置文件，找到 c.NotebookApp.notebook_dir 修改值为本地文件夹目录，打开注释 重新运行 jupyter notebook 即可看到更新

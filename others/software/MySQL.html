# 安装步骤

1.下载地址 下载第一个

MySQL zip下载 (opens new window)

https://dev.mysql.com/downloads/mysql/

2.解压压缩包

3.在解压目录下创建 my.ini 文件，文件内容如下

[mysqld]
# 设置3306端口
port=3306
# 设置mysql的安装目录   ---是你的文件路径---
basedir = F:\\software\mysql-8.0.32
# 设置mysql数据库的数据的存放目录  ---是你的文件路径data文件夹自行创建---
datadir = F:\\software\mysql-8.0.32\data
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。
max_connect_errors=10
# 服务端使用的字符集默认为utf8mb4
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
# 默认使用“mysql_native_password”插件认证
#mysql_native_password
default_authentication_plugin=mysql_native_password
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8mb4
[client]
# 设置mysql客户端连接服务端时默认使用的端口
port=3306
default-character-set=utf8mb4

4.添加环境变量 在系统变量Path中添加 bin目录文件夹

F:\software\mysql-8.0.32\bin

5.初始化 以管理员身份打开cmd，输入 mysqld --initialize --console 命令进行初始化。 输出如下语句,其中root@localhost:后面的**********就是初始密码，初次登陆时要用到。

[Note] [MY-010454] [Server] A temporary password is generated for root@localhost:**********

6.安装 输入 mysqld --install 命令行安装

7.启动 输入 net start mysql 命令启动mysql

8.登录 输入 mysql -u root -p 登录，密码为上面初始化的密码

9.修改密码 输入 ALTER USER USER() IDENTIFIED BY '新密码';, 修改密码

10.退出mysql 输入 quit;

11.关闭服务 输入 net stop mysql
